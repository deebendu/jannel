Jannel - Java Kannel library

The jannel project implements a client for the bearer-box server using the protocol specified by the Kannel project. It is implemented using the Netty IO framework for robustness and performance. The design is based on the well-known cloudhopper-smpp library.

The project is still in beta stages and the API is subject to change. Will provide releases on the central maven repository and try also to provide snapshots using travis.

Usage

Include the dependency of your pom file:

<dependency>
    <groupId>com.github.spapageo</groupId>
    <artifactId>jannel</artifactId>
    <version>0.1.BETA</version>
</dependency>

Using the library works as follows:

ClientSessionConfiguration config = new ClientSessionConfiguration("awesome_box");
config.setHost("localhost");
config.setPort(12000)

JannelClient jannelClient = new JannelClient(2);

ClientSession session = jannelClient.identify(config, mySessionHandler);
